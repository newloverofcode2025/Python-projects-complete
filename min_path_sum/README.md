## Minimum Path Sum

### Description
A function to find the minimum path sum in a grid.

### File
- [min_path_sum.py](min_path_sum.py)
Run the Code
Run the code in VS Code by pressing F5 or by clicking the "Run" button in the top-right corner.
Verify the output:
Minimum path sum in grid [[1, 3, 1], [1, 5, 1], [4, 2, 1]]: 7
Minimum path sum in grid [[1, 2, 3], [4, 5, 6]]: 12
