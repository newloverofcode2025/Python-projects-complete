## Merge k Sorted Lists

### Description
Merge k sorted linked lists into one sorted linked list. Analyze and describe its complexity.
152.2 Example
Input:
lists = [
  1 -> 4 -> 5,
  1 -> 3 -> 4,
  2 -> 6
]
Output:
1 -> 1 -> 2 -> 3 -> 4 -> 4 -> 5 -> 6

A function to merge k sorted linked lists into one sorted linked list.
